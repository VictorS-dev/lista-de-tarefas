# Lista de Tarefas ✅

Site para organizar tarefas com funcionalidades básicas de to-do list.

## 🔧 Funcionalidades

- Adicionar nova tarefa
- Marcar como concluída
- Excluir tarefa
- Armazenamento local (mantém mesmo após fechar o navegador)

## 🖥️ Tecnologias usadas

- HTML
- CSS
- JavaScript
